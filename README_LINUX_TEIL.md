# OwnMessenger Linux-Teil

Diese ZIP enthält nur die Linux-Dateien, die du in dein bestehendes Repository kopieren musst.

## Dateien

```text
server/install.sh
server/.env.linux.example
LINUX_INSTALLATION.md
```

## Einfügen ins Repo

Kopiere die Dateien so in dein GitHub-Projekt:

```text
server/install.sh
server/.env.linux.example
LINUX_INSTALLATION.md
```

## Start unter Linux

```bash
cd server
chmod +x install.sh
./install.sh
```

Das Script installiert beziehungsweise vorbereitet alles, was für den Server unter Linux benötigt wird, und kann danach den Server starten.
